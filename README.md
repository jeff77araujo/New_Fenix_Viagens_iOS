# New_Fenix_Viagens_iOS
Aplicativo desenvolvido para estudos de SwiftUI, como: <br>
 <br>
Criar Views com SwiftUI <br>
Aprender como usar o ScrollView <br>
Implemente navegação entre Views na ação de clique da célula da lista e navegação entre as telas <br>
Criar navegação com TabView <br>
Aprender a fechar Views usando @Environment <br>
Adaptar o layout para iPhone e iPad com @Environment <br>
